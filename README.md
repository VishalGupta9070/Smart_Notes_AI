# 🧠 Smart Notes AI

Smart Notes AI is an Android application that converts unstructured notes into structured, actionable tasks using AI.

---

## 🚀 Features

* 📝 Add messy or free-form notes
* 🤖 AI-powered task extraction
* 🎯 Automatic priority assignment (High, Medium, Low)
* 📅 Deadline suggestions
* 🗂️ View tasks date-wise
* ✅ Mark tasks as completed
* 📆 Filter tasks using calendar

---

## 💡 How It Works

1. User enters unstructured notes
2. App sends input to AI
3. AI processes and returns structured tasks
4. Tasks are displayed with priority & deadlines

---

## 🏗️ Tech Stack

* Kotlin
* Jetpack Compose
* MVVM Architecture
* Retrofit (API calls)
* StateFlow
* Gemini AI API (for task extraction)

---

## 📱 Screens

* Home Screen (Calendar + Task List)
* Add Notes Screen
* Task List Screen

---

## 🧪 Example

**Input:**
Call client tomorrow, finish UI by Friday, go to gym

**Output:**

* Call client → High Priority → Tomorrow
* Finish UI → High Priority → Friday
* Go to gym → Low Priority → Not specified
